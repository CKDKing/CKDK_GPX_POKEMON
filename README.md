# CKDK GPX APP - 寶可夢與最愛位置編輯器

一個高效、美觀、全功能的地圖點資料與軌跡路徑編輯工具，專為寶可夢玩家以及 GPS 虛擬定位管理（如 iToolsBT 與 Android 飛人工具）所設計的單網頁應用程式 (SPA)。

## 🔍 功能總覽

| 功能模組 | 子功能 / 說明 | 資料儲存 / 格式支援 |
| :--- | :--- | :--- |
| **iToolsBT 最愛位置** | 支援多層分類手風琴 (Accordion) 管理、手動排序、跳位、拖曳地圖標記、時區差推算（相較於台北時間） | 本地 `localStorage` / 導入與導出自訂最愛 JSON 檔案 |
| **iToolsBT 路徑編輯** | 路徑節點追加、任意拖曳地圖節點微調、排序、飛人速度設定 | 導入 GPX/JSON、導出 iTools 飛人規格 `.gpx` (實為 JSON) |
| **Android 飛人編輯** | 軌跡點與熱點地標新增/刪除/上下移/輸入編號快速跳位、地圖點擊連續加點、重複座標一鍵去重 (Deduplication) | 導入 GPX/JSON、導出標準 XML 格式 `.gpx` 檔案 |
| **寶可夢查詢** | 內置 1025 隻寶可夢編號、中日英文名、關鍵字搜尋，直連或嵌入 52poke 維基百科詳細資料 | 記憶體快取 (InMemory) / 內嵌 Iframe 顯示 |
| **抓寶工具與相關資訊** | 彙整報寶貝、IV100、PvPokeTW、IV計算機、9db、J-Channel、官方社群等實用工具 | 另開分頁 / 內嵌 iframe 快速瀏覽 |
| **地圖控制核心** | 提供即時經緯度數值與滑鼠跟隨、右鍵快捷複製選單、Dark/Light 傳統圖資及三種 Google 地圖（地圖、衛星、混合）圖資切換 | 整合 Leaflet + Google Map Tile 服務 |

## 💻 環境需求

* **瀏覽器**：建議使用 Google Chrome、Microsoft Edge 或 Safari 等支援最新 HTML5/CSS3 特性與 File System Access API 的現代網頁瀏覽器。
* **網路連線**：由於地圖底圖（Leaflet、CartoDB、Google Map）、Tailwind CSS CDN 以及 Lucide 圖示套件需從 CDN 載入，本系統需要順暢的網路連線以獲得最佳視覺與操作體驗。

## 🚀 安裝與使用說明

本應用為**免安裝單 HTML 檔案**，即開即用，無需搭建任何伺服器或後端資料庫：

1. 下載 `AeroGPX_Editor.html` 至本地資料夾。
2. 按兩下 `AeroGPX_Editor.html`，以瀏覽器開啟即可開始使用。
3. **手機版與響應式支援**：
   * 當偵測到螢幕寬度小於 768px 或於行動裝置開啟時，系統將自動套用手機版佈局。
   * 手機版點擊左上角選單按鈕（Menu）即可展開/收合側邊欄。
   * 所有按鈕、輸入框以及下拉選單（包含「PokemonGO官網 - 其他國家」翻譯連結選單）皆已針對行動裝置觸碰防干擾優化，完美順暢運行。

## 🎨 核心工作模組介紹

### 1. iToolsBT 最愛位置管理
* **手風琴多分類**：預設載入包含 Default、Fight、Asia、North America、Europe 等 10 大實用地點分類。使用者可隨意建立、重新命名或刪除分類，並透過拖曳直接調整分類順序。
* **位置卡片**：每筆最愛位置皆支援手動編輯標題、經緯度，並會**即時根據座標計算出該地的名稱與時區**，算出其相較於台北時間（Asia/Taipei）快/慢幾小時，極大方便計算寶可夢冷卻時間。
* **地圖拖曳**：選中該分類後，地圖上會重繪該分類所有標記。直接在地圖上拖曳標記，即可同步修改並儲存該點經緯度。

### 2. iToolsBT 路徑編輯器
* **軌跡點序列化**：可直接匯入現有飛人路徑軌跡檔，卡片式呈現點順序。
* **地圖微調**：地圖上路徑會以高亮發光線串聯，每個節點皆有數字標記且可直接拖曳。拖曳地圖節點後，左側卡片數值與檔案內容即時更新。
* **飛人專用匯出**：整合一鍵匯出功能，可將路徑轉換為飛人格式的 `.gpx`（內含自訂飛人速度 `speed` 的 JSON 封裝檔）。

### 3. Android 飛人寶可夢軌跡編輯
* **GPX 軌跡管理**：專為 Android GPS Joystick 設計，支援快速新增地標、路線軌跡，調整排序。
* **重複去重 (Deduplication)**：內建重複項目自動偵測，可精準辨識經緯度誤差小於 5 公尺內或名稱完全重複的標記，並提供一鍵清理保留唯一項功能。
* **點擊連續加點**：開啟「地圖點擊新增」狀態後，點擊地圖任意處即可依序快速生成路徑節點。
* **相容匯出**：支援匯出為標準 XML 架構的 `.gpx`。

### 4. 寶可夢與抓寶輔助查詢
* **寶可夢查詢**：側邊欄內建寶可夢全世代資料庫，鍵入編號、中日英文名即可快速篩選，點擊直接於右側面板內嵌載入百科資訊。
* **外嵌/內嵌工具**：將常用查詢網站如 PvPoke、IV CP 對照表與 IV 計算機進行完美整合。
